# Wasteland SA

Wasteland SA é um servidor survival para open.mp inspirado em DayZ, Rust e Escape From Tarkov, desenvolvido do zero com foco em organização, performance e escalabilidade.

## Objetivos

- Survival hardcore
- Loot dinâmico
- Inventário inspirado no Rust
- Sistema de doenças
- Construção de bases
- PvP
- Clãs
- Economia baseada em recursos

## Tecnologias

- open.mp
- Pawn
- MariaDB
- Visual Studio Code
- Git
- GitHub
